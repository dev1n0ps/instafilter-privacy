# Privacy Policy for InstaFilter

InstaFilter is a Chrome browser extension that filters Instagram posts based on user-defined keywords and OCR-detected image text.

## Data Collection
InstaFilter does not collect or store personal data.

## Data Processing
The extension reads content visible on Instagram pages to determine whether posts should be filtered according to the user’s keyword settings.

## OCR Processing
Image text recognition is performed locally in the user's browser using Tesseract.js. No images are uploaded or transmitted to external servers.

## Data Storage
User preferences such as blocked keywords are stored locally using Chrome's storage API.

## Third-Party Services
InstaFilter does not transmit user data to third parties.
